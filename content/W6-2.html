W6-2

擷取個人靜態網頁

在動態的網址上加上變數(?src=)再放上(抽點網站的python語言)所轉換的連結就能夠在動態顯示W6-1抽點網站

後面再加上(&src2)再放上(擷取個人靜態網頁的python語言)所轉換的連結就能夠在動態顯示

連結在此

(https://gist.githubusercontent.com/mdecycu/2c6323eff49b496d1bafd210f3ec9707/raw/035d6948a233beceba7a7974eef0a0a4ef9c17e9/cp2022_w6b_gen_pages_link.py)

from browser import document, html

brython_div1 = document["brython_div1"]
"""
brython_div1 <= html.BUTTON("hello")
brython_div1 <= html.BR()
brython_div1 <= html.A("google", href="https://google.com")
"""

url = "https://mde.tw/studlist/2022fall/1b.txt"
data = open(url).read().split("\n")
mdecp2022 = "https://mdecp2022.github.io/site-"
data = data[1:-1]
for i in data:
    stud = i.split("\t")
    stud_num = stud[0]
    github_acc = stud[1]
    #print(stud_num, github_acc)
    if github_acc == "":
        github_acc = stud_num
    site = mdecp2022 + github_acc
    link = html.A(stud_num, href=site)
    brython_div1 <= link
    brython_div1 <= html.BR()
